# Liberty Web

Simple Web Interface for LibertyBans. Made in Java using the Micronaut Framework.

## Setup
You can find a quick setup guide [here](https://dimitri-bit.github.io/Liberty-Bans-Web/)

## Requirements
- Java 17
- LibertyBans External Database
